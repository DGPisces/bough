# Changelog

All notable changes to Bough are documented here.

## [v1.0.0-rc.1] - 2026-06-02

Initial public prerelease candidate for Bough.

### Added

- macOS notch status surface for supported AI coding tools.
- Bough mascot and supported-tool mascot presentation.
- Usage, music, lyrics, AirDrop, diagnostics, and Settings preview surfaces.
- GitHub Releases DMG install path.
- Public stable-channel automatic update configuration for future stable builds.

### Notes

- Prerelease builds may require manual updates from GitHub Releases.
- Stable `v1.0.0` will be a separate release.
